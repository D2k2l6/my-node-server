Welcome to the Movie API

This API allows you to manage a collection of movies. Below are the available endpoints:

Endpoints

  • GET /movies

    Fetch a list of all movies.

  • POST /movies

    Add a new movie. Example JSON body:

    {
    "title": "Inception",
    "director": "Christopher Nolan",
    "releaseYear": 2010
}
  • PUT /movies/:id

    Update an existing movie by its ID. Example JSON body:

    {
    "title": "Updated Title"
}
  • DELETE /movies/:id

    Delete a movie by its ID.

Response Format

All responses are in JSON format.
